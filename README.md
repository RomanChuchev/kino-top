# [KinoTop](https://romanchuchev.github.io/kino-top/)

## Приложение по поиску фильмов

## Запустить проект

#### `git clone https://github.com/RomanChuchev/kino-top.git`

#### `npm install`

#### `npm start`

#### `npm build`
